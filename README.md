# Sudoku solver on C
Algorithm of filling sudoku on C
## Usage
* Compile project
```
>gcc -o sudoku ft_putchar.c ft_putstr.c ft_strlen.c main.c sudoku.c
```

* Run binary file with inputed sudoku
```
>./sudoku "...36.857" "....8...6" "86...1..9" ".578..64." "9.1.3.5.8" ".86..793." "6..1...85" "5...2...." "128.53..."
```
